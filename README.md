# vis-highlight

A [vis-plugin](https://github.com/martanne/vis/wiki/Plugins/) to highlight patterns.

## How to
 
Press `:hi some-pattern` to highlight text with this Lua pattern.

Disable with `:hi some-pattern off`.

List all patterns with `:hi-ls`.
